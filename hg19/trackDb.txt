    track bigBed
    bigDataUrl data/bigBed/example_bed_3.bigBed
    shortLabel Example bigBed
    longLabel Example of a somple bigBed file in bed 3 format
    type bigBed 3
    parent supertrack

    track bigWig
    bigDataUrl data/bigWig/sine-1000.bedgraph.bigWig

    parent supertrack

    parent supertrack
